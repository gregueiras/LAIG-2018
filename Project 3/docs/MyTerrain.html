Class: MyTerrain

MyTerrain(scene, div, idHeightMap, idTexture, heightScale)

MyTerrain Class, based on Plane, creates a 1x1 plane simulating the relief defined by a height map, heightScale and a texture

Constructor

new MyTerrain(scene, div, idHeightMap, idTexture, heightScale)

Creates an instance of MyTerrain. Calls Plane constructor
Parameters:
Name Type Description
scene CGFscene this instance CGFscene
div Number number of parts to divide the plane, in both axis
idHeightMap String id of the height map texture
idTexture String id of the "real" texture
heightScale Number height scale factor, differentiates between the lower and the upper parts of the image
